To run the project:

1. Open the notebook in Google Colab.
2. Upload "enrolments.csv", "exams.csv", "rooms.csv", "students.csv" and "timeslots.csv" from the dataset into the runtime.
3. Run the notebook.
4. Download "resolved.csv" after execution to obtain the resultant timetable.

NOTES:
*Jupyter can also be used alternatively
*Project_Prototype_1 is for the Nottingham dataset, MSc_Project is the final notebook which uses the Glasgow dataset.
